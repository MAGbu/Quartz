User Documentation



1.     Overview

Team Quartz’s Professor Website Manager is a web based content management system which will allow professors of Boston University’s Department of Computer Science to create websites that are uniform to a department standard. Professors will create a user account on the website manager’s admin page. Once they have created an account professors will be able to input information. The website manager will then take their information which includes things such as a professor’s picture, contact information, and a short biography, and create a website based on it. The websites created will all be of a similar style, the intention of this being that they all conform to department-wide specifications.

 

2.     Creating a user account

·         A user starts by connecting to http://localhost:81/Quartz.

·         If the user does not already have an account on website manager they can click on the “create a new account” link at the bottom of the login box, and follow the rest of the steps. Otherwise the user may choose to just login in. (Please note that a professor is allowed only one account on the website manager).

·         After clicking on create a new account, the user should be redirected to a create new account page.

·         The user should enter the appropriate information in the following dialog boxes:

                                                               i.      Name: the user should enter their first name

                                                              ii.      BU Email: the user should enter their BU email (note, this is the user’s BU login name followed by @bu.edu)

                                                             iii.      Confirm email: the user should re-enter their same email for confirmation

                                                           iv.      Password: the user should enter a password for the site. Note this password must contain at least to special characters (non alphabet characters)

                                                            v.      Confirm password: the user should re-enter the password they have selected for the site, for confirmation.

                                                           vi.      BU ID: The user should enter their 9-digit BU ID. (Uxx-xx-xxxx)

·         After entering all of the appropriate information, and checking it for correctness, the user should hit submit, in the lower right hand corner.

·         After the admin has approved your account, an activation email will be sent within 24 hours.

 

3.     Creating a website

·         Return to the homepage either by clicking login in the upper right hand corner or by going to http://localhost:81/Quartz.

·         After creating an account, a user should log in @ with the username and password that they selected. Note: a user may not create a website until they have created an account.

·         After entering username and password in the appropriate fields a user should hit the login button. This will take the user to the “Website Management Panel.”

·         Once in the “Website Management Panel” a user may enter the following information and have it uploaded to the website:

                                                             i.      Online/Offline: This will publish a user’s website to the internet. A user should leave the offline option selected until they are ready for others to view their website.

 

                                                           ii.      My picture: A user may upload a picture of themselves, by clicking on the browse button, selecting the picture on their computer and then hitting the upload button. The picture should be of the dimensions: 233x350 pixels. Note: it is suggested that professors use their official BU pictures.

 

                                                        iii.      General Information: The user should enter the following information in the appropriate fields.

·         Display Name: The user should enter their name as they would like it to appear on their website

·         Job Title: The user should enter their official BU job title.

·         Office Address: The user should enter their office address, including their office’s room number

·         Telephone: The user should enter their (office) telephone number

·         Fax: The user should enter their office’s fax number

·         Office Hours: The user should enter their office hours using the following format: [DAY TT:TT - TT:TT].

At this point the user should hit the save, to make sure that the information entered in the above fields was not lost.

                                                         iv.      Short Biography: A user may upload a short bio about herself. There are no restrictions on content enforced by the system, thought it is strongly recommended that professors keep this biography professional. There is a fifteen-hundred character limit (1500 characters max). It is recommended that a user copy and paste into this field.

Note: The user should once again hit the save button located in this section to ensure that the information entered in the bio field was not lost.

                                                           v.      Teaching: The user should enter information about courses for which they are instructors:

·         Course Name: The name of the course (as the user would like it to appear)

·         Course Website Link: The URL to the course’s website

·         Short Description: A brief description of the course.

After entering this information the user should select the add new button. If at any time the instructor whishes to remove courses, she can hit the delete all button.

                                                         vi.      Research:

·         Research Summary: The user can enter a synopsis (up to 1500 characters) about the research they are performing, or have performed in the past.

                                                      vii.      Awards

·         Enter a summary of your awards. There is no character limit. HTML formatting can be used.

·         Check the enable box in order to display on website.

                                                    viii.      Projects

·         Enter a summary of your projects. There is no character limit. HTML formatting can be used.

·         Check the enable box in order to display on website.

                                                         ix.      Students

·         Enter the names and information of your students. There is no character limit. HTML formatting can be used.

·         Check the enable box in order to display on website.

                                                           x.      Personal

·         Enter personal information or links. There is no character limit. HTML formatting can be used.

·         Check the enable box in order to display on website.

 

Note: after entering this information the user should once again hit the save button in this area, to ensure that the information they have entered is stored and not lost. Again it is recommended that users copy and paste into this section.

At this point the user should return to the top of the Website Management Panel and select the “online” option and press the apply button, to change the users website from being “under construction” to displaying all of the entered information.
